Interface ICloudFormationContext

Sets a context for CloudFormation.

The context provides essential information for the operation of this library:

  • Valid AWS credentials with rights to run the CloudFormation.
  • Region in which to run the stack.
  • Alternative endpoint URL for CloudFormation service, e.g. if using LocalStack.
  • Logging interface for messages generated within this library.
  • S3 interface to handle the upload of templates or stack policies too large to pass as text.
