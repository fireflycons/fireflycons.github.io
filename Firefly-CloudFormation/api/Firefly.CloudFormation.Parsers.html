Namespace Firefly.CloudFormation.Parsers

Classes

InputFileParser

Base class for all YAML/JSON parser types

ParameterFileParser

Base class for parameter file parsers.

Parameter files contain a JSON or YAML list of parameter structures with fields ParameterKey and ParameterValue. This is similar to aws cloudformation create-stack except the other fields defined for that are currently ignored here. Parameters not supplied to an update operation are assumed to be UsePreviousValue.

ResourceImportParser

Base class for Resource Import file parsers

Resource Import files contain a JSON or YAML list of resource import structures which look like this (JSON) 

[
    {
        "ResourceType":"AWS::DynamoDB::Table",
        "LogicalResourceId":"GamesTable",
        "ResourceIdentifier": {
            "TableName":"Games"
        }
    }
]

See https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/resource-import-existing-stack.html for information on resource import data structure.
