Class ResourceImportParser

Base class for Resource Import file parsers

Resource Import files contain a JSON or YAML list of resource import structures which look like this (JSON) 

[
    {
        "ResourceType":"AWS::DynamoDB::Table",
        "LogicalResourceId":"GamesTable",
        "ResourceIdentifier": {
            "TableName":"Games"
        }
    }
]

See https://docs.aws.amazon.com/AWSCloudFormation/latest/UserGuide/resource-import-existing-stack.html for information on resource import data structure.
