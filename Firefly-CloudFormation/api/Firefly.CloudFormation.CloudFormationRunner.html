Class CloudFormationRunner

Class to manage all stack operations

This is the workhorse of the library, managing interaction with the AWS CloudFormation API. There are no public constructors as a builder pattern is used to create an instance due to the very large number of constructor arguments.

To construct this object, get a builder instance by calling Builder(ICloudFormationContext, String)

ResetStackAsync()

Resets a stack by deleting and recreating it.

This is useful when an attempt to create a new stack fails, leaving an existing stack in ROLLBACK_COMPLETE state, or if you just want to rebuild a stack from scratch. Internally it calls DeleteStackAsync(Func<Boolean>, Func<Boolean>), waits for that to complete, then calls CreateStackAsync().
