Namespace Firefly.CloudFormation

Classes

CloudFormationOperations

Contains methods for operations on stacks that are not related to applying templates

CloudFormationRunner

Class to manage all stack operations

This is the workhorse of the library, managing interaction with the AWS CloudFormation API. There are no public constructors as a builder pattern is used to create an instance due to the very large number of constructor arguments.

To construct this object, get a builder instance by calling Builder(ICloudFormationContext, String)

Interfaces

IAwsClientFactory

Factory interface for the AWS clients used in this library

ICloudFormationContext

Sets a context for CloudFormation.

The context provides essential information for the operation of this library:

  • Valid AWS credentials with rights to run the CloudFormation.
  • Region in which to run the stack.
  • Alternative endpoint URL for CloudFormation service, e.g. if using LocalStack.
  • Logging interface for messages generated within this library.
  • S3 interface to handle the upload of templates or stack policies too large to pass as text.

ICloudFormationRunner

Interface describing CloudFormationRunner. Provided for mocking.

ILogger

Interface to which all messages and stack events are sent.

IS3Util

Interface describing methods that CloudFormation operations can use to upload oversize content (template or policy document) to S3 and to download template content from S3.

If an implementation of this interface is not provided to the CloudFormationBuilder, then an attempt to run with oversize content (e.g. template body > 51,200 bytes) or to refer to a template in S3, then the operation will fail.
